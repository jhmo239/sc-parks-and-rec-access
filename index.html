Public Parks and Recreation Access in South Carolina

Spatial Analysis of Park Accessibility in South Carolina Using 2-Mile Buffer Proximity

Authored by: Jordan Morrison

This map was created using public park and county boundary data to analyze park accessibility across South Carolina. Public park locations were obtained from OpenStreetMap using the QuickOSM plugin in QGIS 4.0.0, extracting features with the tags leisure = park within the state of South Carolina. County boundary data were downloaded from the U.S. Census Bureau’s 2022 TIGER/Line shapefiles. The data were projected using EPSG:2273 (NAD83 / South Carolina ftUS) to ensure spatial consistency across layers. To represent park accessibility, a 2-mile buffer was generated around each park feature. Where parks were represented as polygons, centroid points were created to standardize spatial analysis. The total number of park buffers intersecting each county was then calculated using the Join Attributes by Location (Summary) tool in QGIS. This allowed for a county-level comparison of potential access to public recreational spaces. The final map was symbolized as a choropleth, with graduated colors representing the number of park buffers within each county. Darker shades indicate higher levels of park accessibility. Data sources include OpenStreetMap for park locations and the U.S. Census Bureau TIGER/Line shapefiles for county boundaries:
